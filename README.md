# 💳 Fraud Detection ML Pipeline

## 📊 Overview
Built a machine learning model to detect fraudulent financial transactions using highly imbalanced data.

## 🛠 Tech Stack
- Python
- Scikit-learn
- XGBoost
- SMOTE (Imbalanced-learn)

## ⚙️ Features
- Data preprocessing and imbalance handling
- XGBoost classification model
- Threshold tuning for better fraud detection
- Model evaluation using precision, recall, F1-score
- Feature importance visualization

## 📈 Results
- F1 Score: ~0.80
- Recall: ~0.86 (high fraud detection rate)
- Precision: ~0.74
